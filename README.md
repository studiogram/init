# Formation

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Ajouter un exercice

1. Créez une branche dédiée à l'exercice.
2. Dupliquez le dossier `exercices/exemple` et renommez-le.

## Déploiement

Push sur `main` — GitHub Actions déploie sur GitHub Pages automatiquement.
(Settings/Pages : Source -> Github Actions)
